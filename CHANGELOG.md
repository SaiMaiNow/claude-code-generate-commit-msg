# Change Log

All notable changes to this extension are documented here.

## 0.0.1 - 2026-06-25

- Initial release.
- Generate commit messages from Git diffs through Claude Code CLI.
- Support configurable AI provider/model settings.
- Add custom command provider for alternate AI CLIs.
- Show the generate command only when an AI CLI and Git repository are available.
- Default diff mode to `auto`, preferring staged changes and falling back to unstaged changes.
